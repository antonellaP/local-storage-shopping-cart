Simple product page with JavaScript-powered shopping cart.  
The shopping cart doesn't integrate with any payment menthod, but it retains the selected 
products by using Local Storage.

Web technologies used:
 
- CSS Grid
- Flexbox
- CSS animation and transitions
- CSS gradients
- CSS Clip-path
- HTML5 Local Storage
- Vanilla JS

Project can be previewed on https://codepen.io/antonietta/full/MVedpj/
